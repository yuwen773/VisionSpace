## 三、路由结构

### 3.1 完整路由树

```
/                           → 首页（图片广场）
/user/login                 → 登录
/user/register              → 注册
/add_picture               → 创建/编辑图片
/add_picture/batch         → 批量创建
/picture/:id               → 图片详情
/add_space                 → 创建/编辑空间
/my_space                  → 中转页 → 私人空间
/space/:id                 → 空间详情
/space_analyze             → 空间分析
/search_picture            → 搜索图片
/spaceUserManage/:spaceId  → 空间成员管理
/admin/*                   → 管理后台（需admin权限）
```

### 3.2 权限系统

| 权限类型 | 说明 |
|---------|------|
| admin角色 | 访问管理后台 `/admin/*` |
| 空间角色 | viewer/editor/admin |
| 操作权限 | picture:upload/edit/delete 等 |

## 六、优化建议清单

### 6.1 高优先级优化

| 问题 | 当前状态 | 建议 |
|------|----------|------|
| **请求拦截器** | 拦截器挂在全局 axios | 应该用 myAxios.interceptors |
| **环境变量** | 硬编码 URL | 使用 .env 文件管理 |
| **响应处理** | 404 返回 undefined | 统一错误处理，返回 Promise.reject |
| **类型安全** | 使用 any 类型 | 明确类型定义 |

### 6.2 中优先级优化

| 优化项 | 预期收益 |
|--------|----------|
| Token 管理 | 提升安全性 |
| 按需引入 Ant Design | 减少包体积 ~30% |
| 添加加载状态 | 提升用户体验 |
| 路由守卫优化 | 避免重复获取用户信息 |

### 6.3 低优先级优化

| 优化项 | 说明 |
|--------|------|
| 代码格式化钩子 | 使用 husky + lint-staged |
| 虚拟滚动 | 大数据场景优化 |
| 组件懒加载 | 对大型组件使用 defineAsyncComponent |

## 七、动态主题切换功能 ⭐新增

### 7.1 功能概述

实现了完整的动态主题切换系统，支持：

- ✅ 用户下拉菜单切换
- ✅ 浮动按钮快速切换
- ✅ localStorage 持久化
- ✅ 系统偏好自动检测
- ✅ 无刷新动态加载

### 7.2 新增文件

```
src/
├── composables/
│   └── useTheme.ts              # 主题管理逻辑
└── components/
    └── ThemeToggleButton.vue    # 浮动切换按钮
```

### 7.3 修改文件

| 文件 | 修改内容 |
|------|----------|
| `src/main.ts` | 移除静态引入，使用动态主题加载 |
| `src/components/GlobalHeader.vue` | 添加用户菜单主题切换 |
| `src/layout/BasicLayout.vue` | 添加浮动按钮组件 |

### 7.4 使用方式

**方式一：用户菜单切换**
```
点击右上角头像 → 主题切换 → 选择主题
```

**方式二：浮动按钮切换**
```
点击右下角圆形按钮，一键切换主题
```

### 7.5 主题管理 API

```typescript
// 使用主题管理
import { useTheme } from '@/composables/useTheme'

const {
  currentTheme,      // 当前主题
  isLoading,         // 加载状态
  THEMES,            // 主题配置
  initTheme,         // 初始化主题
  toggleTheme,       // 切换主题
  setTheme,          // 设置指定主题
  getCurrentThemeInfo // 获取当前主题信息
} = useTheme()
```

## 八、关键特性说明

### 8.1 WebSocket 协同编辑

- **文件**: `src/utils/pictureEditWebSocket.ts`
- **功能**: 团队空间下图片实时编辑同步
- **特性**:
  - 编辑状态锁定
  - 操作广播（旋转、缩放）
  - 进入/退出编辑状态管理

- 分类分析（双Y轴柱状图）
- 大小分析（饼图）
- 标签分析（词云图）
- 排行分析
- 使用分析
- 用户分析

### 8.3 图片处理功能

- **图片裁剪**: vue-cropper 组件
- **AI 扩图**: 任务创建 + 状态轮询
- **以图搜图**: 图片搜索功能
- **批量上传**: 支持关键词批量抓取
