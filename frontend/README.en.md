# VisionSpace

#### 介绍

VisionSpace 是一个企业级智能协同云图库管理系统，提供图片上传、管理、分享等功能。支持多主题切换（深空黑 / 波普风格），采用现代化的 UI 设计。

#### 技术栈

- **前端框架**: Vue 3 + TypeScript
- **构建工具**: Vite 6
- **UI 组件库**: Ant Design Vue 4
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **图表**: ECharts + vue-echarts
- **图片处理**: Vue-Cropper
- **样式**: Less + CSS Variables
